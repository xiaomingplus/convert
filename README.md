# 进制转换Alfred workflow

方便快捷的转换进制。

## 下载链接

<https://github.com/xiaomingplus/convert/releases/download/v1.0.0/convert.alfredworkflow>

## 如何使用

- 10进制转别的进制直接输入```conv 123```
- 16进制转别的进制直接输入```conv 0x1a```
- 8进制转别的进制直接输入```conv 0o123```
- 2进制转别的进制直接输入```conv 0o101```
